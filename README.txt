Учебный проект на базе фреймворка yii (ver 1.1.14 )

Inna Zeyda 

1) Скрипт для создания базы: ./protected/data/schema.mysql.sql
2) В файле ./protected/config/main.php необходимо изменить параметры соединения с сервером mysql на свои
3) Директорий ./images/ и ./tmp/ должны быть доступны для записи пользователем, от имени которого 
   запущен веб-сервер.
4) Недостаток (поздно обнаруженный) :
   Изображения хранятся не в базе данных, а в файловой системе и названы <id>.jpg. По этой причине при
   замене картинки автомобиля из-за кэширования браузером отбражается старая картинка. Необходимо генерировать
   имена файлов картинок случайным образом и хранить ссылку на файл в базе данных. Либо хранить сами изображения
   там.
5) В проекте не предусмотрен интерфейс для редактирования таблицы "тип кузова". Редактирование таблицы "цвета"
   (добавление новых цветов) возможно на странице добавления/изменения модели автомобиля.
